픽셀그래픽(비트맵그래픽) 툴

픽셀 - 모니터 화면을 구성하는 가장 작은 단위

레이어 잘 선택돼있는지 확인!

작업중 레이어 확인 필수

선택영역지정 -> 픽셀영역 지정

레이어보다 선택영역이 우선순위가 높음
어떤 작업이든 선택영역이 지정되어있으면 영역안에서의 작업을 우선 처리함

선택영역은 레이어 바꿔도 유지되므로 선택 해제 해야됨~ ctrl+D

전경색 채우기 alt+delete
배경색 채워넣기 Ctrl+delete

일러스트레이터와 동일, 이미지 가져올 때 마다 새로운 레이어 생성됨

레이어 복사 ctrl+j 

레이어 안 색깔이 칠해진 픽셀부분 선택해주는 단축기 ctrl + 레이어창 섬네일 클릭

free transform
ctrl + t

자유롭게 크기조절 및 회전 가능
shift 누르면 비율 깨고 변형 가능
ctrl 누르면 포인트 개별적 변형 가능

canvas 사이즈 조정

absolute 방식
- 캔바스의 크기를 수치로 나타냄. 일러와 동일하게 계산식 사용하여 수치조정 가능

relative 방식
- 상대값 사용하여 감소,증가 값 입력

-----------

clone stamp 도구
-픽셀 복사 붙여넣기 하는 도구

1. alt+클릭 하여 클릭한 지점을 기준으로 픽셀데이터 복사
2. 복사한 픽셀을 클릭/드래그 하여 사용
(자연스럽게 soft-round브러쉬 사용)

선택영역 조정

select -> modify 메뉴

geather - 선택한 영역의 가장자리에서 뭉개짐 정도를 의미
수치가 커질수록 더 넓게 뭉개지며 선택 됨
(픽셀선택툴을 사용하기 전에 상단 옵션창에서 feather값 확인)

expand - 선택한 영역의 가장자리에서 바깥쪽으로 확장하여 픽셀을 더 선택함
contranct - 가장자리에서 안쪽으로 축소하여 픽셀 선택함
